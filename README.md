# faa-sam-demo

This is the repository needed to run the FAA demo during the AWS re:Invent 2025 workshop.

## Getting Started

1. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Follow the instructions in the AWS Studio workshop.

## Solution

The complete solution can be found in the `solution` directory.

## Important Notes

- Make sure to follow the AWS Studio workshop instructions carefully.
- The `solution` directory contains the final implementation of the demo.

For any questions or issues, please refer to the workshop materials or contact the workshop organizers.
